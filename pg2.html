How to Scrape Tables from a Web page

Pandas library - last edited April 1, 2021

Pandas has a method to read through an HTML web page, and extract tables from that page. To scrape the tables from an HTML page, we can use .read_html() method. To load the table into a Pandas DataFrame, we pass in the index of the table in brackets. If there is only one table on the page, the index is 0.

import pandas as pd
table=pd.read_html('https://akhadianharnowo.github.io/Dr.H/x1.html')
table[0]

If the page has multiple tables, the .read_html() method will list all tables it could find. We only need to slice the table that we want to load to the DataFrame by specifying the index.

# to load the second table in the output of pd.read_html() method
table[1]

We can use the len() method to learn how many tables Pandas found on an HTML page.

len(table)
