How to Scrape Data from the Internet

HTML Basics - April 1, 2021

HTML, stands for Hyper Text Markup Language, is the standard markup language for creating web pages. HTML defines the structure of a web page and shows its content. HTML consists of a series of elements which tell browser how to display the content correctly. Elements are declared by using tags: a start tag and an end tag. Web page content is placed in between those two tags.

At the very basic, an HTML file needs:
1. A document type declaration:<!DOCTYPE html>
2. The HTML document tags:starts with <html> and ends with </html>
3. The page content tags:starts with <body> and ends with </body>

When we type in a URL in the search address bar on an internet browser, we are sending an HTTP request for a webpage, and the webpage server will send back a response to our computer. Our browser will render the page based on the HTML elements consist in the file.

Requests
Requests is an HTTP library for Python that pulls the raw content of a webpage, which is similar to opening an HTML file on Notepad or code editors. To pull the webpage content from a specific URL, we use the .get() method, then we store the content in variable called response. For example:

import requests
response=requests.get("https://akhadianharnowo.github.io/Dr.H/basic.html")

Beautiful Soup

a Data Scraping Tool - April 2, 2021

Beautiful Soup is a Python library for screen scraping, that is, pulling data out of markup languages, such as HTML and XML. Beautiful Soup helps with pulling particular content from a webpage, parsing the file, removing the (HTML) markups & tags, and saving the information.

To create a Beautiful Soup document, which is called 'soup', we call the constructor: BeautifulSoup() and pass in two parameters:"url.html" and "html.parser". For example:

from bs4 import BeautifulSoup
soup = BeautifulSoup("a web page", 'html.parser')
