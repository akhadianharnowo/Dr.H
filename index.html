Data Discovery and Management (DA 348) is the first of four required courses in the Business Data Analytics major at Washburn University School of Business. The major provides students with the skill sets needed to transform data into useful information and insights for data-driven decision making. You can read more detailed information about the program from Washburn’s Data Analytics page.

DA 348 can also be taken as an elective course or as a part of a minor. Feel free to contact me if you have questions about the course or the program.

Data Analytics Major

The following courses are required for the business data analytics major:


DA 348

Data Discovery and Management

This course is the first step in the data analytics program. In this course, students will identify and manipulate data that will provide actionable information to solve business problems.
In this course, students will learn Python programming

Prerequisites:
EC 211, BU 248, BU 250

DA 358

Data Methods and Warehousing

Students will learn methods to process a variety of data types (unstructured and semi-structured) and use technologies that convert, analyze and store large volumes of data. Unstructured and semi-structured data will be converted into information useful for problem solving.

Prerequisite: DA 348

DA 368

Data Mining and Modeling

Students will learn technologies that can be used to discover relationships among data. These relations can be used to create models used to predict or classify new data.

Prerequisite: DA 348

DA 478

Data Analytics Applied - Practicum

Students will apply the data analytics process including data discovery, transformation, organization and modeling to a real-world project and effectively communicate the solutions.

Prerequisites:
DA 358, DA 368

Why DA 348?

Learn Python

In DA 348, students will learn Python programming for data analytics. Python programming opens the door to many opportunities:
1. Python advances your problem-solving skills and is relatively easy to learn

2. Advances business/ data analytics career or get higher salary

  • Python deals with any types of data: numerical, categorical, geographical, text, images, audio, video, etc. (i.e., structured, semi-structured, unstructured data) from various sources
  • Python can perform statistics, predictive analytics, visualization, simulation (prescriptive analytics), among others
  • Python can process large datasets
  • Python can automate many business processes
  • Many (big) companies require python skills

3. Pivots to a different career such as web developer, software engineer, data science, etc.
